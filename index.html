SECTION 01

Introduction to C Programming

C is a procedural systems language used in operating systems, embedded firmware, drivers, and performance-critical software. It teaches direct control over memory and execution flow, which is why it remains a core interview language for software engineering roles.

#include <stdio.h>

int main(void) {
    printf("Hello, C!\n");
    return 0;
}

SECTION 02

Data Types in C

Every variable in C has a concrete data type that defines memory size and legal operations. Correct type choice improves correctness, memory usage, and runtime performance.

#include <stdio.h>

int main(void) {
    int age = 21;
    float cgpa = 8.5f;
    char grade = 'A';
    printf("%d %.1f %c\n", age, cgpa, grade);
    return 0;
}

SECTION 03

Control Flow

Control flow statements decide which blocks execute and when they execute. Mastering branching helps avoid edge-case bugs and improves program readability.

#include <stdio.h>

int main(void) {
    int x = 7;
    if (x % 2 == 0) {
        printf("Even\n");
    } else {
        printf("Odd\n");
    }
    return 0;
}

SECTION 04

Loops

Loops perform repeated work with predictable structure. They are essential for traversal, counting, and iterative algorithm implementation.

#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    return 0;
}

SECTION 05

Arrays

Arrays store fixed-size sequential elements in contiguous memory. They are foundational to search, sort, matrix, and buffer-based interview questions.

#include <stdio.h>

int main(void) {
    int arr[3] = {10, 20, 30};
    printf("%d\n", arr[1]);
    return 0;
}

SECTION 06

Strings in C

In C, strings are null-terminated character arrays and must be managed carefully. Correct handling of boundaries and terminators prevents memory bugs and security issues.

#include <stdio.h>

int main(void) {
    char s[] = "C Tutor";
    printf("%s\n", s);
    return 0;
}

SECTION 07

Pointers

Pointers hold memory addresses and let you work directly with memory and dynamic data structures. They are critical for linked lists, trees, dynamic allocation, and systems-level coding.

#include <stdio.h>

int main(void) {
    int x = 42;
    int *p = &x;
    printf("%d\n", *p);
    return 0;
}
